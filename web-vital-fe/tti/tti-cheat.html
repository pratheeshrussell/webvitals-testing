TTI Cheat Testing

NOTE: This is just for demo, and is not a way to improve TTI.
TTI measures how long it takes for a web page to be fully interactive. This occurs at a point when the last Long Task finishes and is followed by 5 seconds of inactivity on the main thread.
Based on that logic this page should have a very good TTI, eventhough a long task is scheduled after 5.5seconds.
